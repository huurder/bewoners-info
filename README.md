# bewoners-info

Static GitHub Pages form for the Barnstijn Beheer B.V. tenant initiative.

## Publish checklist

1. Replace `PUBLIC_KEY` in `app.js` with the real armored PGP public key.
2. Set `FORM_ENDPOINT` in `app.js` to the production form endpoint.
3. If using Web3Forms, replace `FORM_ACCESS_KEY` in `app.js`.
4. Keep `servicekosten.png`, `index.html`, `styles.css`, and `app.js` in the published GitHub Pages branch.
